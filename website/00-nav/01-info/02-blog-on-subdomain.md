# Host your own blog on a subdomain

If you wish, you can start your own thematic blog/site on a subdomain like your\_{last_name, nick_name}.oracledba.net. The topic is Oracle technologies.

This can be useful for storing your knowledge, which you can always refer to.

I haven't looked into the matter, but it seems to me that ready-made solutions like blogger, wordpress can be integrated (maybe it costs some money, I don't know).

jekyll (what this site runs on), octopress and any html site can be hosted on github for free.
You can connect the angular.js library to an HTML page, which allows you to create headers and footers for html sites, and basically nothing more is needed.

Of course, there should be no warez, viruses, or any materials that violate laws or harm me.

P.S. If oracledba.net seems long, you can use a shorter name, for example your\_{last_name, nick_name}.oradba.net or other available options.

P.S. Indians (and not only them) promote their blogs, then write a letter to Oracle asking them to review their blogs and award them the Oracle ACE achievement and as a result add them to the catalog on the Oracle website.

The lower the number in this catalog, the cooler it is. Maybe this gives some advantages in something.
