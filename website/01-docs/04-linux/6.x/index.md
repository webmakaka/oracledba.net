# Installation of Oracle Linux 6.X

<br/>

## [Installation of Oracle Linux 6.7 x86 64 bit]:

Unfortunately, in version 6.x of RedHat distributions, the ability to select certain parameters was removed from the text installation procedure (or I just don't know how to configure the network, partition disks as I want, select the necessary packages, etc.).

Press Tab and enter the command linux text

At startup, you need to press tab and append linux text. Otherwise, the installation starts in graphical mode and eventually the GUI will also be installed on the server.
