# Oracle DBA Certification

<br/>

If you want to work at an international level rather than a local one and receive a more decent salary for your work, you need to demonstrate your professional training. Oracle admins need to obtain OCP. To do this, you need to pass exams 1Z0-051, 1Z0-052, 1Z0-053.
