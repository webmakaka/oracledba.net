# [Oracle RAC 12.1 Installation on Oracle Linux 6.7 (ISCSI + ASM)]: Distributives and additional software

<br/>

The latest versions of Oracle DB and Critical Patch Updates are available to commercial subscribers with an active technical support contract. If you have a support contract, you can download the database distributives directly from support.oracle.com.

If you do not have a contract, simply registering on the Oracle website will not be enough to download the latest versions, even for evaluation.

Nevertheless, links to the latest version distributives are provided below.

There are no activation keys. For development, the distributives can be used free of charge, but for their intended purpose (production), a license is required.
