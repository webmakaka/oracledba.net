# Upgrade and Migration Labs for Oracle 12c

**Links:**

<br/>

**Description:**

<br/>

This virtual machine is an image in Oracle VirtualBox hypervisor format.

To run the VM, you need the Oracle VirtualBox hypervisor: https://www.virtualbox.org/wiki/Downloads

The virtual machine contains installed Oracle Database versions 11.2.0.4 and 12.1.0.2 on Oracle Linux 5.11 x86-64bit, and is intended for lab work on training migration to Oracle 12c from previous versions.

The lab work also covers consolidation of Oracle databases using the new container database architecture (pluggable database).
