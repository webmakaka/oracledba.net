# Oracle DBA Library - Books

[SQL and PL/SQL materials are here](//plsql.ru/library/)

<br/>

If you are just starting to learn the basics of working with Oracle databases, I would recommend the following:

<br/>

### Oracle DBA's Reference Book

Sam Alapati: Oracle DataBase 11g Administrator's Guide. The quality of the book is excellent.

This is the book that lay on my desk for quite a long time when server administration was within my competence. Part of the site's materials are taken from it.

The book may seem quite difficult.

<br/>

### Core

[Jonathan Lewis] Oracle Core. Internal workings for database administrators and developers / trans. from English by A. N. Kiselev – M.: DMK Press, 2015. – 372 p.: ill. ISBN 978-5-97060-169-3

<br/>

~Question (a person works at a company that uses Oracle database servers):<br/><br/>

Well, I completed the comprehensive course "Oracle Database 11g Administration" at Specialist.. what's next? I'm still afraid to take on the DB.<br/><br/>

~Answer:<br/><br/>

You need practice.

<br/><br/>

If possible, try to set aside a computer and deploy a database on it from some Oracle backups. Or
clone an already running one. Maybe one of the DBAs will help.
(if possible of course, when I worked I had that
opportunity.).

<br/><br/>

You can also use some standard
schemas, but it's better to use something close to real data that you
will be working with.

<br/><br/>

Well, then you should follow all the steps from the Administration 2 course on
database recovery, i.e., recovery after losing a control file, after
losing a redo log, import/export, backups using RMAN,
rollback to a specific date, Upgrade to the next version.
